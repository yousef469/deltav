# JARVIS - Cognitive Automation Standalone Demo

This is a standalone, performance-optimized demo of the JARVIS v24.5 Cognitive Pipeline. It is built as an independent project using **React 19**, **Tailwind CSS v4**, and **Framer Motion**.

1. **Navigate to the demo folder:**
   ```bash
   cd standalone-demo
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in your browser:**
   The demo will be available at `http://localhost:5173` (or the next available port).
