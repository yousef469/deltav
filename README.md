# Delta-V Portfolio

A high-end, interactive portfolio website built with React 19, Tailwind CSS v4, Framer Motion, and Three.js.

## How to Run

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Open in browser:**
    Navigate to `http://localhost:5173` (or the port shown in the terminal).

## Tech Stack
*   **Framework:** React 19 + Vite
*   **Styling:** Tailwind CSS v4
*   **Animation:** Framer Motion
*   **3D Graphics:** Three.js
*   **AI:** Google Gemini API (for VectorAI chatbot)
